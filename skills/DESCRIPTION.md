# memory-decay skills

Teaches the agent how to use persistent memory with natural decay effectively.

## Skills

| Skill | Trigger | Purpose |
|-------|---------|---------|
| install | User wants to install memory-decay | Full setup guide |
| update | User wants to update memory-decay | Version check + update |
| remember | After learning something important | How to store memories |
| recall | Starting a task, user references past | How to search memories |
| forget | User wants to delete a memory | Safe deletion workflow |
| memory-status | Checking system health | Diagnostics |
